Automated popular science article generation with text simplification

About the Project

Scientific publications are a vital tool for spreading knowledge, but their complexity often limits accessibility for the general public. As research advances, simplified content becomes increasingly important for informed decision-making in fields like medicine and technology.

In this project, we developed an LLM-based system to simplify scientific texts while preserving accuracy and logical flow using Natural Language Processing (NLP) techniques.

What We Did:

  • Selected a T5 model and performed fine-tuning
  • Created a customized training dataset
  • Built an automatic preprocessing pipeline
  • Replaced the attention mechanism and implemented an importance mask
  • Developed a web application with an intuitive user interface

Training Details:

  • Fine-tuned the model using the processed corpus
  • Training included 8 epochs and took ~40 hours
  • Real-time metric monitoring with Wandb
  • Gradual improvement in loss throughout training
